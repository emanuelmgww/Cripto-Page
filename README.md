# Cripto Page

Este projeto foi criado com o objetivo de explorar o design e a experiência de uso de uma plataforma inspirada em corretoras de criptomoedas.
A proposta é apresentar informações de forma clara, moderna e acessível, focando em visualização do mercado, organização de dados e interface intuitiva.

O site não realiza investimentos reais e tem finalidade educacional e de portfólio, servindo como prática de UI/UX e desenvolvimento front-end.
